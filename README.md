## 🎯 What mindsight does

mindsight helps you track objects in a video and map where a person is looking. It is made for research work in cognitive science, neuroscience, psychology, and eye-tracking studies.

Use it to:
- detect objects in a scene
- track gaze points over time
- compare gaze with object location
- review results for research and data analysis

## 🧭 How to use mindsight

### 1. Load your study files
Open your project and add:
- video clips
- images
- gaze data
- object labels

### 2. Set the detection area
Mark the part of the frame you want to analyze. This helps mindsight focus on the right scene.

### 3. Run object detection
Start object detection to find items in each frame. The app will mark detected objects for review.

### 4. Add gaze data
Import gaze points or eye-tracking results. mindsight will line up gaze data with the scene.

### 5. Review intersections
Check where gaze points land on detected objects. This helps you study attention and viewing behavior.

### 6. Export results
Save the output for later analysis in your research workflow.

## 🧩 Common problems

### The app does not open
- check that the file finished downloading
- unzip the file if needed
- run the app from the extracted folder
- try opening it as administrator

### The screen looks empty
- confirm that you loaded a video or image
- check that your project folder is correct
- make sure the file is not damaged

### Gaze data does not line up
- check the time stamp format
- make sure the video frame rate matches your data
- confirm that the gaze file uses the right screen size

### Object labels look wrong
- review the detection settings
- use a clearer source video
- try a short test file first
